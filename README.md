🐾 PAWS & CLAWS – Premium Pet E-commerce Website with Cart, Wishlist & Smart Search
------------------------------------------------------------------------------------
Live Demo: https://niaj-morshed.github.io/paws-pet-shop/


🚀 Overview:
-------------
A fully responsive and interactive pet product e-commerce website built using modern frontend
technologies. The project focuses on delivering a smooth user experience with advanced UI/UX 
features like cart management, wishlist system, live search, and animated interactions.


✨ Key Features:
-----------------
1. 🛍️ Product System
   * Dynamic product rendering from JavaScript data
   * Category-based filtering (Dogs, Cats, Birds, etc.)
   * Responsive grid layout (mobile, tablet, desktop)

2. 🛒 Cart System (Fully Functional)
   * Add to cart functionality
   * Quantity increase/decrease
   * Auto total price calculation
   * Cart drawer (slide animation)
   * LocalStorage support (data saved after refresh)

3. ❤️ Wishlist System
   * Add/remove products to favorites
   * Animated heart interaction (pulse effect)
   * Separate wishlist drawer
   * Persistent data using LocalStorage

4. 🔍 Smart Search System
   * Live search suggestions while typing
   * Click suggestion to filter products instantly
   * Enter key search support
   * Auto hide suggestion dropdown

5. 🌙 Dark / Light Mode
   * Toggle between themes
   * Fully styled using CSS variables
   * Smooth transition effect

6. ⚡ Skeleton Loading (Modern UI Feature)
   * Fake loading animation before rendering products
   * Improves user experience like real-world apps
  
7. 🔔 Toast Notification System
   * Custom animated notifications
   * Used for: Add to cart, Wishlist actions, Order confirmation

8. 💳 Checkout System
   * Interactive checkout form
   * Input validation (name, phone, address)
   * Payment method selection (bKash / Card)
   * Order confirmation with UI feedback

9. ⭐ Ratings & Reviews UI
   * Star rating display
   * Random review count generation
   * Enhances product realism

10. 🎨 Advanced UI/UX Design
    * Glassmorphism navbar
    * Gradient overlays
    * Hover animations (zoom, reveal)
    * Micro-interactions (bounce, pulse)
    * Custom scrollbar styling

11. 📲 Fully Responsive Design
    * Mobile-first layout
    * Horizontal scroll category buttons
    * Adaptive grid system



🛠️ Technologies Used:
----------------------
1. Core
   * HTML5 (Structure)
   * CSS3 (Styling & Animations)
   * JavaScript (Logic & Interactivity)

2. UI Techniques
   * CSS Variables (Theme system)
   * Flexbox & Grid Layout
   * Keyframe Animations
   * Responsive Media Queries

3. Functionality
   * DOM Manipulation
   * Event Handling
   * LocalStorage (Data persistence)

4. External Resources
   * Google Fonts
   * Font Awesome Icons
   * Unsplash Images
